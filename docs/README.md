# Introduction

Welcome to the **zcrAI** platform documentation.

zcrAI is a modern **Security Operations Center (SOC) Dashboard** designed to streamline threat detection, investigation, and response.

## Key Features

*   ** Unified Dashboard**: Visualize your security posture in real-time.
*   ** Automated Detection**: Sigma-based rules to catch threats automatically.
*   ** Case Management**: Integrated Kanban board for efficent incident triage.
*   ** Automation **: Playbooks to block IPs and isolate hosts instantly.
