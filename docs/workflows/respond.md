# Respond (Playbooks)

**Page**: `Automation > Playbooks` (`/playbooks`)

## Overview
Playbooks allow you to automate response actions to contain threats quickly and reduce manual work.

## Common Playbooks
*   **Block IP on Firewall**: Automatically adds a malicious IP to your firewall's blocklist.
*   **Isolate Endpoint**: Quarantines a compromised host from the network (via EDR integration).
*   **Disable User Account**: Locks an Active Directory account showing suspicious activity.

## Execution
*   **Automatic**: Playbooks can be set to run automatically when specific rules trigger.
*   **Manual**: Analysts can trigger playbooks manually from the Case Details page.
